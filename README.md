# MongoDB Assignment 1

## Database: Library

## Collection: Books

## Description
This assignment demonstrates MongoDB CRUD operations using a Library database.

## Operations Done
1. Insert 20 documents
2. Find books where Price > 300
3. Reduce Price by 100 where Stock > 100
4. Find returned books where Price > 500
5. Add Discount field where Stock > 100
